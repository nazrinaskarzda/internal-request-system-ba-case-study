# Initiation Phase

This phase focuses on defining the business problem, identifying stakeholders, and establishing the project scope.

---

## Key Activities

* Defined the business problem and need
* Identified and analyzed stakeholders
* Established project scope
* Defined roles and responsibilities (RACI)
